# _Example_Program — Active Program Folder

**Status:** Example folder — ready to populate with your program
**Context Files Location:** Level_2_Strategy / G5_Active_Programs
**Workflow:** Program planning and execution tracking

---

## What This Folder Contains

Each active program gets its own subfolder here. A program is any sustained initiative that cuts across multiple areas of the business — a funding sprint, a content distribution campaign, an academic submission track, an outreach program, etc.

## Folder Structure

```
PGM-XXX_[Program_Name]/
├── PGM-XXX_Brief.md          — Program definition, success criteria, timeline
├── PGM-XXX_Log.md            — Running log of activity, decisions, milestones
└── [supporting files]         — Any assets, research, or working docs for this program
```

## How to Use This Structure

1. Load your SuperContext and Personal Preferences
2. Tell the AI: "I need to set up a new active program for [program name]. Create the program brief."
3. Use the program brief to align the initiative with your OKRs in Level_2_Strategy/G0_Strategic_Infrastructure/
