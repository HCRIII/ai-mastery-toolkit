**What:** Specialized AI subagents that handle specific types of tasks autonomously

**Purpose:** Agents allow you to spawn specialized subagents for complex tasks like codebase exploration, implementation planning, or multi-step research. Each agent type has specific tools and capabilities optimized for its purpose.

---

## Available Agent Configurations

This folder contains example agent configurations you can learn from and customize:

- `bash-agent.md` - Command execution specialist
- `explore-agent.md` - Fast codebase exploration
- `plan-agent.md` - Implementation planning architect
- `general-purpose-agent.md` - Complex multi-step tasks
- `research-deep-dive.md` - Deep research agent example

All configurations are in Markdown with copy-pasteable prompt templates.
