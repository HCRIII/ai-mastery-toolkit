## Purpose
Generate new content, documents, code, visualizations, and structured outputs from scratch or based on specifications.

## Categories

### Business_Documents/
Create PRDs, design docs, user stories, LOE documents, business proposals
- **When to use:** Product planning, project scoping, business documentation

### Content/
Create summaries, newsletters, emails, essays, stories, aphorisms
- **When to use:** Content marketing, writing, storytelling, communication

### Educational/
Create quizzes, flashcards, reading plans, learning materials
- **When to use:** Teaching, course creation, study materials, knowledge transfer

### Security/
Create security updates, threat models, detection rules, threat scenarios
- **When to use:** Security documentation, threat modeling, SOC operations

### Technical/
Create code, git commits, commands, patterns, technical documentation
- **When to use:** Software development, DevOps, technical writing

### Visualizations/
Create diagrams, visualizations, graphs, concept maps, video chapters
- **When to use:** Visual communication, data presentation, knowledge mapping

## Common Use Cases
- Generate product requirement documents
- Create educational content and quizzes
- Build security documentation and threat models
- Develop technical code and documentation
- Design visual diagrams and concept maps
