## Navigation Tips

- Use **Ctrl+F** (or Cmd+F on Mac) to search for specific prompt names
- Prompts are organized by primary category (numbered 00-15)
- Subcategories are grouped by function or domain
- Each entry includes the relative path for easy navigation
- Descriptions indicate the primary purpose of each prompt

## Usage Conventions

- Prompts beginning with `t_` typically indicate tools or utilities
- Prompts beginning with `create_` are for generating new content
- Prompts beginning with `analyze_` are for analyzing existing content
- Prompts beginning with `extract_` are for pulling specific information
- Core system prompts (00-) are foundational and session-oriented
