## Purpose
Assess quality, rate content, and provide structured evaluations with scoring.

## Categories

### Content/
Rate content quality, value, "wow per minute" metrics
- **When to use:** Content quality assessment, value evaluation

### Quality/
Rate AI responses, AI results, judge output quality
- **When to use:** Evaluating AI outputs, quality control

### Assessment/
Check agreements, find logical fallacies, label and rate, find hidden messages
- **When to use:** Critical analysis, contract review, argument evaluation

## Common Use Cases
- Evaluate content before publication
- Rate AI-generated outputs
- Check contracts for issues
- Find logical flaws in arguments
- Assess information quality
