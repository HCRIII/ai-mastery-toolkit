## Description
Summarizes AI chat prompts by describing the primary function, unique approach, and expected output in a concise paragraph. The summary is focused on the prompt's purpose without unnecessary details or formatting.

## When to Use
- When you need to accomplish the task described above
- As part of a larger workflow or analysis pipeline
- When you need structured, consistent output for this type of task

## Input Requirements
- Provide the content or data to be processed
- Content should be relevant to the prompt's purpose
- Can be text, code, data, or other formats as appropriate

## Output Format
- Structured markdown output
- Organized sections based on prompt specifications
- Formatted for readability and further processing

## Usage Notes
- Review the system.md file for detailed instructions
- Outputs follow the Fabric pattern format
- Can be chained with other prompts for complex workflows
