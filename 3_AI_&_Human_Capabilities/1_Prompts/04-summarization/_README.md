## Purpose
Condense content to essential points, main ideas, and key takeaways.

## Categories

### General/
General-purpose summaries, micro-summaries, debates, lectures
- **When to use:** Quick content review, general summarization needs

### Research/
Summarize academic papers, research newsletters, prompts
- **When to use:** Research synthesis, academic content review

### Business/
Summarize meetings, board meetings, conference calls
- **When to use:** Business documentation, meeting notes, executive summaries

### Technical/
Summarize git changes, pull requests, code diffs
- **When to use:** Development workflow, code review, change documentation

## Common Use Cases
- Create executive summaries of long documents
- Summarize research papers quickly
- Document meeting outcomes
- Review code changes and pull requests
- Create micro-summaries for social sharing
