## Overview

This category contains prompts specifically designed for the Cognitive Architecture Academy launch and operations. These prompts help create curriculum materials, presentations, quizzes, reading plans, and coaching content aligned with the Academy's learning philosophy and brand.

---

## Key Features

- **Mastery-Based Learning**: Content designed around mastery principles, not time-based completion
- **Flow State Integration**: Kotler's flow framework applied to learning design
- **Deliberate Practice**: Ericsson's principles embedded in curriculum structure
- **Brand Alignment**: All content reflects Academy brand voice and positioning

---

## Available Prompts

### Curriculum Development
- **create_academy_curriculum** - Generate curriculum modules and learning sequences
- **create_academy_reading_plan** - Design reading plans with spaced repetition
- **create_academy_quiz** - Create assessment tools aligned with mastery learning

### Content Creation
- **create_academy_presentation** - Build presentation materials for Academy content

### Coaching Integration
- **analyze_coaching_approach** - Evaluate and refine coaching methodologies

## Usage Guidelines

**Before Using These Prompts:**
1. Review Academy brand guidelines (Doc 10 in Life & Business Context)
2. Load relevant SuperContext for personalization
3. Consider target audience (high-performing professionals and executives)

**Quality Standards:**
- Research-backed (cite Kotler, Ericsson, Attia where relevant)
- Mastery-focused (not time-based)
- Action-oriented (not passive consumption)
- Brand-aligned (professional depth + warmth)
