## Description
Design mastery-based learning modules for Cognitive Architecture Academy grounded in deliberate practice, Flow State science, and transformational learning principles.

## When to Use
- Creating new Academy curriculum modules
- Designing course content for founding members
- Developing learning experiences for executive transitions
- Structuring mastery-based progression paths

## Key Features
- **SuperContext Integration**: Loads teaching philosophy, frameworks, and Academy structure
- **Research-Grounded**: Based on Flow/Kotler, Deliberate Practice/Ericsson, Longevity/Attia
- **Protective Design**: Prevents burnout while enabling ambitious growth
- **Mastery-Based**: True skill development, not just completion tracking

## Input Requirements
- Curriculum topic or domain to design
- Target mastery level desired
- Student context (if known)

## Output Format
- Comprehensive module design document
- Structured learning units
- Deliberate practice exercises
- Mastery assessment rubrics
- Flow State integration strategies
- Anti-fragility safeguards
