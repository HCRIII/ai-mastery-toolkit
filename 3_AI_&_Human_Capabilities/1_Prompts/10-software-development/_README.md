## Purpose
Specialized development prompts for coding, git workflows, code review, and development tooling.

## Categories

### Code_Analysis/
Analyze code quality and structure (see 01_Analysis/Technical and 05_Explanation)
- Review code, explain code functionality

### Code_Creation/
Generate code and features (see 02_Creation/Technical)
- Create coding projects, features, patterns

### Git_Workflow/
Git operations and commit management (see 02_Creation/Technical)
- Create git commit messages, manage pull requests

### Development_Tools/
Development utilities and agent-based tools
- ask_uncle_duke - Coordinate AI agents for software solutions
- suggest_gt_command - Suggest git-tree commands
- suggest_moltbot_command - Suggest Moltbot automation commands

## Note
Most software development prompts are organized in their functional categories:
- **Code analysis** → 01_Analysis/Technical/, 05_Explanation/
- **Code creation** → 02_Creation/Technical/
- **Git workflows** → 02_Creation/Technical/, 04_Summarization/Technical/

This category contains specialized development tools and cross-references.

## Common Use Cases
- Generate code projects and features
- Review and improve code quality
- Create git commit messages
- Explain complex codebases
- Coordinate multi-agent development
