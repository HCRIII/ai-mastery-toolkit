## Purpose
Pull specific information, insights, patterns, and structured data from unstructured content.

## Categories

### Insights/
Extract wisdom, insights, ideas, patterns, alpha (novel information)
- **When to use:** Learning from content, finding key insights, pattern recognition

### Structured_Data/
Extract references, recipes, features, domains, video IDs, metadata
- **When to use:** Data extraction, metadata collection, structured information

### Content_Elements/
Extract predictions, questions, instructions, quotes, characters, jokes
- **When to use:** Content analysis, finding specific elements, research

### Security/
Extract proof-of-concepts, CTF writeups, security recommendations
- **When to use:** Security research, vulnerability analysis, threat extraction

## Common Use Cases
- Extract key insights from podcasts, books, articles
- Pull structured data from documents
- Find predictions and questions in content
- Extract security indicators and recommendations
- Collect references and citations
