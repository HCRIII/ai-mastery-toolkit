## Purpose
Specialized security prompts for threat analysis, incident response, security tooling, and defensive operations.

## Categories

### Threat_Analysis/
Security-focused analysis prompts (see 01_Analysis/Security for main prompts)
- Analyze malware, incidents, threat reports, email headers

### Security_Creation/
Security content creation (see 02_Creation/Security for main prompts)
- Create threat models, detection rules, security updates

### Incident_Response/
Respond to and document security incidents
- Incident analysis and reporting

### Security_Tools/
Security tooling and rule creation
- Write Nuclei templates, Semgrep rules, HackerOne reports
- Greybeard secure prompt engineering

## Note
Many cybersecurity prompts are organized in their functional categories:
- **Analysis prompts** → 01_Analysis/Security/
- **Creation prompts** → 02_Creation/Security/
- **Extraction prompts** → 03_Extraction/Security/

This category contains specialized tools and cross-references.

## Common Use Cases
- Create detection rules (Sigma, Nuclei, Semgrep)
- Analyze security incidents
- Build threat models
- Write security reports
- Develop security content
