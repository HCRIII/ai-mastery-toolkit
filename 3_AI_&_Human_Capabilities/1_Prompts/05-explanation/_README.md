## Purpose
Make complex topics, code, mathematics, and documentation understandable and clear.

## What's Here
- **explain_code** - Explain code functionality, security tool output, configurations
- **explain_docs** - Improve and restructure documentation for clarity
- **explain_math** - Clarify mathematical concepts in engaging ways
- **explain_project** - Summarize project documentation into clear sections
- **explain_terms** - Create glossaries with definitions, analogies, context
- **ai** - Interpret questions deeply and provide insightful answers
- **provide_guidance** - Offer guidance and advice on topics
- **raw_query** - General question answering

## When to Use
- Understanding unfamiliar code or technical systems
- Clarifying mathematical concepts
- Improving documentation readability
- Creating glossaries for complex terminology
- Explaining project structure and purpose

## Common Use Cases
- Onboard to new codebases
- Learn new technical concepts
- Create user-friendly documentation
- Build glossaries for teams
- Explain complex systems simply
