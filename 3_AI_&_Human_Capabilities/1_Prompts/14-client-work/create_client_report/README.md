## Description
Creates professional client status reports that communicate progress transparently, surface issues proactively, manage expectations effectively, and maintain trust through clear, honest communication.

## When to Use
- Weekly or bi-weekly client project updates
- Phase completion reports
- Status updates before client meetings
- When significant changes or blockers occur
- Monthly progress summaries for retainer clients

## Key Features
- SuperContext integration for accurate progress tracking
- Positive framing with honest transparency
- Clear visual indicators (, •, ) for scannability
- Proactive issue and risk surfacing
- Client action items clearly stated
- Business value framing for technical work
- Professional, ready-to-send format

## Input Requirements
- Work completed this period
- Current status and active work
- Blockers or challenges encountered
- Client action items needed
- Upcoming work plan
- Timeline status

## Output Format
Professional status report (1-2 pages) with:
- Executive summary (150 words max)
- Progress and wins this period
- Current status and health indicator
- Challenges, blockers, and risks
- Client action items with deadlines
- Upcoming work preview
- Timeline update (if applicable)
- Next check-in agenda

## Usage Notes
- Lead with wins and progress (positive frame)
- Be honest about challenges (no surprises later)
- Surface issues early while they're still manageable
- Clearly state what you need from client
- Use visual indicators for quick scanning
- Frame technical work in business value
- Professional but conversational tone
- Ready to send directly to client
