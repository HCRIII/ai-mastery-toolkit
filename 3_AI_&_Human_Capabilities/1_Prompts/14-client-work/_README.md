## Overview

This category contains prompts specifically designed for client work, including coaching session documentation, client reports, feedback analysis, and project requirements. All prompts are designed to produce professional, polished outputs ready for client delivery.

---

## Key Features

- **SuperContext Integration**: Most prompts load client-specific context from SuperContext for personalized outputs
- **Professional Tone**: All outputs are client-ready with appropriate formality and polish
- **Business Value Framing**: Focus on outcomes and value, not just technical details
- **Scope Management**: Built-in considerations for scope and expectations

---

## Available Prompts

### Client Documentation
- **create_client_report** - Generate comprehensive client reports with findings and recommendations
- **create_client_prd** - Create Product Requirements Documents for client projects
- **summarize_client_session** - Document coaching/consulting sessions with action items

### Analysis & Feedback
- **analyze_client_feedback** - Process and synthesize client feedback into actionable insights

## Usage Guidelines

**Before Using These Prompts:**
1. Ensure SuperContext is current with client information
2. Review scope and deliverable requirements
3. Consider timeline and client expectations

**Quality Standards:**
- Ready to send directly to clients (no cleanup needed)
- Business value clearly articulated
- Clear next steps and owners identified
- Professional tone maintained throughout

## Integration Points

**With SuperContext:**
- Layer 1: Client information and project context

- Layer 3: Communication preferences and brand voice
