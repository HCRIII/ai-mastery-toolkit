## Example: Market Research Project

**Project Name:** "AI Tools Market Research 2026"

### Custom Instructions

```
You are conducting market research and competitive analysis for the AI productivity tools market.

## Research Approach
- Data-driven: Ground insights in provided research, not speculation
- Multi-perspective: Consider user, business, and technical viewpoints
- Synthesis-focused: Connect patterns across sources
- Actionable: Provide clear implications and recommendations
- Cite sources: Reference specific documents when making claims

## Analysis Structure
1. **Gather**: Extract relevant data from all sources
2. **Organize**: Group by themes, patterns, trends
3. **Analyze**: Apply frameworks (SWOT, Porter's Five Forces, etc.)
4. **Synthesize**: Connect insights across sources
5. **Recommend**: Provide clear, actionable conclusions

## Frameworks to Use
- **SWOT Analysis**: Strengths, Weaknesses, Opportunities, Threats
- **Porter's Five Forces**: Competitive dynamics
- **PESTLE**: Political, Economic, Social, Technological, Legal, Environmental factors
- **Value Chain Analysis**: Where value is created and captured
- **Trend Analysis**: Patterns over time, growth trajectories

## Output Style
- Start with executive summary (3-5 key findings)
- Use data to support every claim
- Provide specific examples from sources
- Include both quantitative and qualitative insights
- End with clear recommendations

## When You Don't Know
If information isn't in the provided documents: "This information isn't available in the current research. To answer this, we'd need [specific data/sources]."
```

### Documents to Include

1. **Industry Reports** - Market size, growth rates, key players
2. **Competitor Analysis** - Product features, pricing, positioning
3. **Customer Research** - Surveys, interviews, feedback, reviews
4. **Financial Data** - Revenue, funding, market share (if available)
5. **Trend Reports** - Emerging technologies, shifts in demand
6. **Academic Research** - Relevant studies, papers, meta-analyses
7. **News & Articles** - Recent developments, announcements
8. **Internal Data** - Your own metrics, performance data (if applicable)

### Planning Prompts

**Create Research Framework:**
```
Help me create a research framework for [topic].

What are the key dimensions I should analyze?
What data sources would be most valuable?
What frameworks would be most appropriate?
What questions should drive the research?
```

**Draft Research Plan:**
```
Based on the documents in this project, create a research plan that includes:
1. Key questions we can answer with available data
2. Gaps in our current information
3. Recommended analysis approaches
4. Expected deliverables and timeline
```

**Generate Analysis Template:**
```
Create a template for analyzing [specific aspect: competitor, trend, customer segment].

Include:
- What to look for in each data source
- How to structure findings
- What metrics matter most
- How to synthesize insights
```

**Competitive Analysis:**
```
Analyze our top 5 competitors across these dimensions:
- Product features and capabilities
- Pricing and business model
- Target customers and positioning
- Strengths and weaknesses
- Market share and trajectory

Create a comparison matrix and identify key insights.
```

**Trend Synthesis:**
```
Based on all sources, identify:
1. Major trends shaping [industry/market]
2. Supporting evidence for each trend
3. Implications for [specific stakeholder]
4. Recommended actions
```

## Key Research Project Tips

- **Version your data** - Date-stamp documents so you know how current they are
- **Track sources** - Always know where each insight comes from
- **Update regularly** - Add new research as it becomes available
- **Separate facts from interpretation** - Make clear what's data vs. analysis
- **Test your conclusions** - Ask Claude to challenge your assumptions
- **Create reusable templates** - Build analysis frameworks you can apply repeatedly
- **Export key findings** - Save important analyses as separate documents for easy reference
