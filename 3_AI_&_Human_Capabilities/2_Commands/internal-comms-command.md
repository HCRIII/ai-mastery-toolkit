# /internal-comms Command

**Created:** 2026-02-13
**Last Updated:** 2026-02-13
**Purpose:** Command reference for Internal Comms capability
**Status:** Final

---

**Command:** `Use the internal-comms skill`

**What it does:** Helps write all kinds of internal communications using company-preferred formats including status reports, leadership updates, newsletters, FAQs, incident reports, and project updates.

---

## How to Invoke

**Explicit command:**
```
Use the internal-comms skill
```

**Automatic triggers:**
- "status reports"
- "leadership updates"
- "3P updates"
- "company newsletters"
- "FAQs"
- "incident reports"
- "project updates"
- Other internal communications requests

---

## When to Use

Use this command whenever asked to write internal communications:
- Status reports
- Leadership updates
- 3P (third-party) updates
- Company newsletters
- FAQs
- Incident reports
- Project updates
- Other company communications

**Provides:**
- Company-preferred formats
- Appropriate tone and structure
- Internal communication best practices

---

## Example Invocations

**Example 1: Explicit command**
```
Use the internal-comms skill to write a status report
```

**Example 2: Automatic trigger (status report)**
```
Create a status report for the Q2 migration project
```

**Example 3: Automatic trigger (leadership update)**
```
Write a leadership update about our product launch
```

**Example 4: Automatic trigger (incident report)**
```
Draft an incident report for yesterday's outage
```
