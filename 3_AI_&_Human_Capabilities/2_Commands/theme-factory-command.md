# /theme-factory Command

**Created:** 2026-02-13
**Last Updated:** 2026-02-13
**Purpose:** Command reference for Theme Factory capability
**Status:** Final

---

**Command:** `Use the theme-factory skill`

**What it does:** Toolkit for styling artifacts with themes - applies to slides, docs, reports, HTML landing pages, etc. Provides 10 preset themes with colors/fonts, or can generate new themes on-the-fly.

---

## How to Invoke

**Explicit command:**
```
Use the theme-factory skill
```

**Automatic triggers:**
- Requests to style artifacts
- Apply themes to content
- Make visual improvements to artifacts

---

## When to Use

Use this command for:
- Styling slides, docs, reports, HTML landing pages
- Applying one of 10 preset themes
- Generating a new custom theme on-the-fly
- Consistent visual styling across artifacts

**Artifacts can be:**
- Slides
- Documents
- Reports
- HTML landing pages
- Any other artifacts needing visual theming

---

## Example Invocations

**Example 1: Explicit command**
```
Use the theme-factory skill to style my presentation
```

**Example 2: Automatic trigger (style artifact)**
```
Apply a professional theme to this report
```

**Example 3: Automatic trigger (visual improvement)**
```
Make this slide deck look more polished
```

**Example 4: Custom theme**
```
Create a warm, inviting theme for these workshop slides
```
