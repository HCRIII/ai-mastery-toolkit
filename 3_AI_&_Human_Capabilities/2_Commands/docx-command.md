# /docx Command

**Created:** 2026-02-13
**Last Updated:** 2026-02-13
**Purpose:** Command reference for Docx capability
**Status:** Final

---

**Command:** `Use the docx skill`

**What it does:** Creates, reads, edits, and manipulates Microsoft Word (.docx) documents with professional formatting, tracked changes, comments, and complex layouts.

---

## How to Invoke

**Explicit command:**
```
Use the docx skill
```

**Automatic triggers:**
- "Word doc", "word document", ".docx"
- "report", "memo", "letter", "template"
- Professional documents with formatting
- Tables of contents, headings, page numbers, letterheads

**File uploads:**
- Upload .docx file → command activates automatically

---

## When to Use

Use this command when you need to:
- Create professional Word documents
- Extract or reorganize content from .docx files
- Insert or replace images in documents
- Perform find-and-replace in Word files
- Work with tracked changes or comments
- Convert content into polished Word format

**Do NOT use for:**
- PDFs (use `/pdf` instead)
- Spreadsheets (use `/xlsx` instead)
- Google Docs
- General coding tasks unrelated to documents

---

## Example Invocations

**Example 1: Explicit command**
```
Use the docx skill to create a business proposal
```

**Example 2: Automatic trigger (mention Word)**
```
Create a Word document about our Q2 strategy
```

**Example 3: Automatic trigger (mention report)**
```
Make me a report on customer feedback analysis
```

**Example 4: File upload**
- Upload a .docx file
- Command activates automatically
- Ask to edit, extract, or analyze the document
