# /create-shortcut Command

**Created:** 2026-02-13
**Last Updated:** 2026-02-13
**Purpose:** Command reference for Create Shortcut capability
**Status:** Final

---

**Command:** `Use the create-shortcut skill`

**What it does:** Creates or manages shortcuts that can be run on demand or automatically on a schedule for task automation.

---

## How to Invoke

**Explicit command:**
```
Use the create-shortcut skill
```

**Automatic triggers:**
- Requests to create shortcuts
- Task automation requests
- Scheduled automation needs

---

## When to Use

Use this command to:
- Create shortcuts that run on demand
- Create shortcuts that run automatically on a schedule
- Automate repetitive tasks
- Set up workflow automations

**Shortcuts can be:**
- **On-demand:** Run when you trigger them
- **Scheduled:** Run automatically at specified times

---

## Example Invocations

**Example 1: Explicit command**
```
Use the create-shortcut skill
```

**Example 2: On-demand shortcut**
```
Create a shortcut to generate my weekly status report
```

**Example 3: Scheduled shortcut**
```
Set up a shortcut to backup my files every Friday at 5pm
```

**Example 4: Workflow automation**
```
Make a shortcut that checks email, summarizes messages, and adds action items to my todo list
```
