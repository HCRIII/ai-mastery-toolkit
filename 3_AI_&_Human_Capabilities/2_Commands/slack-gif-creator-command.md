# /slack-gif-creator Command

**Created:** 2026-02-13
**Last Updated:** 2026-02-13
**Purpose:** Command reference for Slack Gif Creator capability
**Status:** Final

---

**Command:** `Use the slack-gif-creator skill`

**What it does:** Creates animated GIFs optimized for Slack with proper constraints, validation tools, and animation concepts.

---

## How to Invoke

**Explicit command:**
```
Use the slack-gif-creator skill
```

**Automatic triggers:**
- "make me a GIF of X doing Y for Slack"
- "animated GIFs for Slack"
- Requests for Slack-specific GIF creation

---

## When to Use

Use this command when users request:
- Animated GIFs optimized for Slack
- Custom Slack emoji animations
- GIFs with specific Slack constraints (file size, dimensions)

**Provides:**
- Slack-specific constraints
- Validation tools
- Animation concepts

---

## Example Invocations

**Example 1: Explicit command**
```
Use the slack-gif-creator skill to make a celebration GIF
```

**Example 2: Automatic trigger (specific request)**
```
Make me a GIF of a rocket launching for Slack
```

**Example 3: Automatic trigger (Slack mention)**
```
Create an animated GIF for Slack showing "great job"
```

**Example 4: Automatic trigger (animation)**
```
Generate a GIF of confetti for our team Slack
```
