# /skill-creator Command

**Created:** 2026-02-13
**Last Updated:** 2026-02-13
**Purpose:** Command reference for Skill Creator capability
**Status:** Final

---

**Command:** `Use the skill-creator skill`

**What it does:** Guides creation of effective custom skills that extend Claude's capabilities with specialized knowledge, workflows, or tool integrations.

---

## How to Invoke

**Explicit command:**
```
Use the skill-creator skill
```

**Automatic triggers:**
- User wants to create a new skill
- User wants to update an existing skill

---

## When to Use

Use this command when users want to:
- Create a new skill
- Update an existing skill
- Extend Claude's capabilities
- Package specialized knowledge as a skill
- Create custom workflows
- Build tool integrations

**This skill should be used:**
- **BEFORE creating your first custom skill**
- When you need guidance on skill structure
- When you want to ensure best practices

---

## Example Invocations

**Example 1: Explicit command**
```
Use the skill-creator skill
```

**Example 2: Automatic trigger (create skill)**
```
Help me create a new skill for analyzing customer feedback
```

**Example 3: Automatic trigger (update skill)**
```
I want to update my meeting notes skill
```

**Example 4: Automatic trigger (guidance)**
```
Guide me through creating a skill for our company workflow
```
