# /pptx Command

**Created:** 2026-02-13
**Last Updated:** 2026-02-13
**Purpose:** Command reference for Pptx capability
**Status:** Final

---

**Command:** `Use the pptx skill`

**What it does:** Creates, reads, edits, and manipulates Microsoft PowerPoint (.pptx) presentations with slides, layouts, speaker notes, and visual content.

---

## How to Invoke

**Explicit command:**
```
Use the pptx skill
```

**Automatic triggers:**
- "deck", "slides", "presentation"
- Any mention of .pptx filename
- "pitch deck", "slide deck"

**File uploads:**
- Upload .pptx file → command activates automatically

---

## When to Use

Use this command whenever a .pptx file is involved — as input, output, or both:
- Creating slide decks, pitch decks, or presentations
- Reading, parsing, or extracting text from .pptx files
- Editing, modifying, or updating existing presentations
- Combining or splitting slide files
- Working with templates, layouts, speaker notes, or comments

**Triggers whenever:**
- User mentions "deck," "slides," "presentation"
- User references a .pptx filename
- A .pptx file needs to be opened, created, or touched

---

## Example Invocations

**Example 1: Explicit command**
```
Use the pptx skill to create a pitch deck
```

**Example 2: Automatic trigger (mention slides)**
```
Make me slides for the product launch
```

**Example 3: Automatic trigger (mention deck)**
```
Create a deck about our Q1 results
```

**Example 4: File upload**
- Upload a .pptx file
- Command activates automatically
- Extract content, edit slides, or analyze presentation
