# /canvas-design Command

**Created:** 2026-02-13
**Last Updated:** 2026-02-13
**Purpose:** Command reference for Canvas Design capability
**Status:** Final

---

**Command:** `Use the canvas-design skill`

**What it does:** Creates beautiful visual art in .png and .pdf formats using design philosophy for posters, designs, and static visual pieces.

---

## How to Invoke

**Explicit command:**
```
Use the canvas-design skill
```

**Automatic triggers:**
- User asks to create a poster
- User asks to create art, design, or static piece
- Visual design requests

---

## When to Use

Use this command when the user asks to:
- Create posters
- Create pieces of art
- Create designs
- Create other static visual pieces

**Important:** Creates **original** visual designs. Never copies existing artists' work to avoid copyright violations.

---

## Example Invocations

**Example 1: Explicit command**
```
Use the canvas-design skill to create a concert poster
```

**Example 2: Automatic trigger (poster)**
```
Create a poster for our community event
```

**Example 3: Automatic trigger (design)**
```
Design a visual for the product launch
```

**Example 4: Automatic trigger (art)**
```
Make me art representing growth and innovation
```
