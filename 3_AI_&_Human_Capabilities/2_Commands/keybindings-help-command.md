# /keybindings-help Command

**Created:** 2026-02-13
**Last Updated:** 2026-02-13
**Purpose:** Command reference for Keybindings Help capability
**Status:** Final

---

**Command:** `Use the keybindings-help skill`

**What it does:** Helps customize keyboard shortcuts by modifying ~/.claude/keybindings.json to add chord bindings, rebind keys, and personalize controls.

---

## How to Invoke

**Explicit command:**
```
Use the keybindings-help skill
```

**Automatic triggers:**
- "rebind ctrl+s"
- "add a chord shortcut"
- "change the submit key"
- "customize keybindings"
- Any keyboard customization requests

---

## When to Use

Use this command when the user wants to:
- Customize keyboard shortcuts
- Rebind keys
- Add chord bindings (multi-key sequences)
- Modify ~/.claude/keybindings.json
- Change submit key or other controls

---

## Example Invocations

**Example 1: Explicit command**
```
Use the keybindings-help skill
```

**Example 2: Automatic trigger (rebind)**
```
Rebind Ctrl+S to save my session
```

**Example 3: Automatic trigger (chord)**
```
Add a chord shortcut: Ctrl+K then D to open docs
```

**Example 4: Automatic trigger (submit key)**
```
Change the submit key from Ctrl+Enter to just Enter
```
