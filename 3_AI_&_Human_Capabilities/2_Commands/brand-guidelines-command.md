# /brand-guidelines Command

**Created:** 2026-02-13
**Last Updated:** 2026-02-13
**Purpose:** Command reference for Brand Guidelines capability
**Status:** Final

---

**Command:** `Use the brand-guidelines skill`

**What it does:** Applies Anthropic's official brand colors and typography to any artifact that may benefit from having Anthropic's look-and-feel.

---

## How to Invoke

**Explicit command:**
```
Use the brand-guidelines skill
```

**Automatic triggers:**
- "brand colors or style guidelines"
- "visual formatting"
- "company design standards"
- Requests for Anthropic branding

---

## When to Use

Use this command when:
- Applying Anthropic branding to artifacts
- Need brand colors or style guidelines
- Visual formatting with company standards
- Company design standards apply

**Provides:**
- Official Anthropic brand colors
- Typography standards
- Visual formatting guidelines
- Consistent company look-and-feel

---

## Example Invocations

**Example 1: Explicit command**
```
Use the brand-guidelines skill for this presentation
```

**Example 2: Automatic trigger (brand colors)**
```
Apply Anthropic brand colors to this slide deck
```

**Example 3: Automatic trigger (company standards)**
```
Format this document using company design standards
```

**Example 4: Automatic trigger (branding)**
```
Make this look like an Anthropic document
```
