# /algorithmic-art Command

**Created:** 2026-02-13
**Last Updated:** 2026-02-13
**Purpose:** Command reference for Algorithmic Art capability
**Status:** Final

---

**Command:** `Use the algorithmic-art skill`

**What it does:** Creates algorithmic art using p5.js with seeded randomness and interactive parameter exploration for generative art, flow fields, and particle systems.

---

## How to Invoke

**Explicit command:**
```
Use the algorithmic-art skill
```

**Automatic triggers:**
- "creating art using code"
- "generative art"
- "algorithmic art"
- "flow fields"
- "particle systems"

---

## When to Use

Use this command when users request:
- Creating art using code
- Generative art
- Algorithmic art
- Flow fields
- Particle systems

**Important:** Creates **original** algorithmic art. Never copies existing artists' work to avoid copyright violations.

---

## Example Invocations

**Example 1: Explicit command**
```
Use the algorithmic-art skill to create a flow field visualization
```

**Example 2: Automatic trigger (generative art)**
```
Create generative art with organic patterns
```

**Example 3: Automatic trigger (flow fields)**
```
Make a flow field with blue and purple gradients
```

**Example 4: Automatic trigger (code)**
```
Generate art using code with particle systems
```
