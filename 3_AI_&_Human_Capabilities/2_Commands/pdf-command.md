# /pdf Command

**Created:** 2026-02-13
**Last Updated:** 2026-02-13
**Purpose:** Command reference for Pdf capability
**Status:** Final

---

**Command:** `Use the pdf skill`

**What it does:** Reads, extracts, manipulates, and creates PDF files including text/table extraction, merging, splitting, forms, watermarks, encryption, and OCR.

---

## How to Invoke

**Explicit command:**
```
Use the pdf skill
```

**Automatic triggers:**
- User mentions .pdf file
- User asks to produce a PDF
- Any PDF-related task

**File uploads:**
- Upload .pdf file → command activates automatically

---

## When to Use

Use this command whenever the user wants to do anything with PDF files:
- Reading or extracting text/tables from PDFs
- Combining or merging multiple PDFs into one
- Splitting PDFs apart
- Rotating pages
- Adding watermarks
- Creating new PDFs
- Filling PDF forms
- Encrypting/decrypting PDFs
- Extracting images
- OCR on scanned PDFs to make them searchable

---

## Example Invocations

**Example 1: Explicit command**
```
Use the pdf skill to merge these three contracts
```

**Example 2: Automatic trigger (mention PDF)**
```
Extract all the tables from this financial report PDF
```

**Example 3: Automatic trigger (create PDF)**
```
Convert this document to PDF format
```

**Example 4: File upload**
- Upload one or more .pdf files
- Command activates automatically
- Extract, merge, split, or manipulate PDFs
