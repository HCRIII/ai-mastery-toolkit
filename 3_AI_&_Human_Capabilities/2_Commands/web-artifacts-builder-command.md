# /web-artifacts-builder Command

**Created:** 2026-02-13
**Last Updated:** 2026-02-13
**Purpose:** Command reference for Web Artifacts Builder capability
**Status:** Final

---

**Command:** `Use the web-artifacts-builder skill`

**What it does:** Creates elaborate, multi-component HTML artifacts for claude.ai using modern frontend technologies (React, Tailwind CSS, shadcn/ui) with state management and routing.

---

## How to Invoke

**Explicit command:**
```
Use the web-artifacts-builder skill
```

**Automatic triggers:**
- "complex artifacts"
- "multi-component web app"
- Tasks requiring shadcn/ui components

---

## When to Use

Use this command for:
- Complex artifacts requiring state management
- Multi-component applications with routing
- Artifacts needing shadcn/ui components

**Do NOT use for:**
- Simple single-file HTML/JSX artifacts
- Basic web pages without complex state

**Use regular artifact creation instead for simple cases.**

---

## Example Invocations

**Example 1: Explicit command**
```
Use the web-artifacts-builder skill to create an interactive dashboard
```

**Example 2: Automatic trigger (complex artifact)**
```
Build a complex web app with multiple tabs and data visualization
```

**Example 3: Automatic trigger (multi-component)**
```
Create a multi-component artifact with user authentication flow
```

**Example 4: Automatic trigger (shadcn/ui)**
```
Make an artifact using shadcn/ui components for the interface
```
