# /mcp-builder Command

**Created:** 2026-02-13
**Last Updated:** 2026-02-13
**Purpose:** Command reference for Mcp Builder capability
**Status:** Final

---

**Command:** `Use the mcp-builder skill`

**What it does:** Guides creation of high-quality MCP (Model Context Protocol) servers that enable LLMs to interact with external services through well-designed tools in Python (FastMCP) or Node/TypeScript (MCP SDK).

---

## How to Invoke

**Explicit command:**
```
Use the mcp-builder skill
```

**Automatic triggers:**
- "building MCP servers"
- "integrate external APIs or services"
- MCP server development tasks

---

## When to Use

Use this command when:
- Building MCP servers to integrate external APIs or services
- Creating tools in Python (FastMCP)
- Creating tools in Node/TypeScript (MCP SDK)
- Connecting LLMs to external services

---

## Example Invocations

**Example 1: Explicit command**
```
Use the mcp-builder skill to create a GitHub integration
```

**Example 2: Automatic trigger (MCP server)**
```
Build an MCP server for our company API
```

**Example 3: Automatic trigger (integration)**
```
Create an MCP integration with Slack
```

**Example 4: Automatic trigger (external service)**
```
Help me connect to the Notion API using MCP
```
