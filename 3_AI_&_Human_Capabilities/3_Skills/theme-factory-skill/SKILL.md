# Theme Factory Skill - Artifact Styling

## What This Skill Does

Toolkit for styling artifacts with themes including slides, docs, reports, HTML landing pages with 10 preset themes (colors/fonts) or custom theme generation.

## When to Use This Skill

**Automatic triggers:**
- "style artifact", "apply theme"
- Theme or styling mentions for artifacts

**Use when you need:**
- Consistent visual styling
- Professional appearance
- Branded artifacts
- Themed presentations
- Styled documents
- Visual cohesion

## How to Invoke

**Explicit invocation:**
```
Use the theme-factory skill
```

**Trigger words (automatic):**
- "Apply a theme to..."
- "Style this artifact..."
- "Make this look professional..."

## Capabilities

- **10 Preset Themes:** Ready-to-use color/font combinations
- **Custom Themes:** Generate new themes on-the-fly
- **Artifact Types:** Slides, docs, reports, HTML
- **Color Schemes:** Professional palettes
- **Typography:** Font pairings
- **Consistent Styling:** Cohesive appearance

## Example Usage

### Example 1: Style Presentation
**You say:** "Apply a professional blue theme to my presentation about quarterly results"

**What happens:**
- Skill reviews preset themes
- Selects appropriate blue theme
- Applies colors and fonts
- Ensures readability
- Returns styled presentation

### Example 2: Custom Theme
**You say:** "Create a warm, inviting theme with orange and cream colors for my workshop slides"

**What happens:**
- Skill generates custom theme
- Creates orange/cream palette
- Selects complementary fonts
- Applies to slides
- Returns themed artifact

### Example 3: Report Styling
**You say:** "Make this technical report look professional and modern"

**What happens:**
- Skill selects appropriate theme
- Applies professional styling
- Ensures consistency
- Returns polished report

## Tips for Best Results

**Specify:**
- Desired mood (professional, playful, serious, warm)
- Color preferences
- Industry or context
- Artifact type

**For custom themes:**
- Describe desired feeling
- Mention brand colors if applicable
- Reference style inspirations
