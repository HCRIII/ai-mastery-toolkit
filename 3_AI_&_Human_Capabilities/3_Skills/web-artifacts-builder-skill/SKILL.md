# Web Artifacts Builder Skill - Complex Web Artifacts

## What This Skill Does

Creates elaborate, multi-component HTML artifacts for claude.ai using modern frontend technologies (React, Tailwind CSS, shadcn/ui) with state management and routing.

## When to Use This Skill

**Automatic triggers:**
- "complex artifact", "multi-component web app"
- Artifacts requiring shadcn/ui components

**Use when you need:**
- Complex interactive web applications
- Multi-page artifacts with routing
- Advanced state management
- shadcn/ui component library
- Interactive dashboards
- Web tools and calculators

**Note:** For simple single-file HTML/JSX artifacts, use regular artifact creation instead.

## How to Invoke

**Explicit invocation:**
```
Use the web-artifacts-builder skill
```

**Trigger words (automatic):**
- "Create a complex web artifact..."
- "Build a multi-component app..."
- "Make an interactive dashboard..."

## Capabilities

- **React:** Full React component system
- **Tailwind CSS:** Utility-first styling
- **shadcn/ui:** Pre-built component library
- **State Management:** Complex state handling
- **Routing:** Multi-page navigation
- **Interactivity:** Advanced user interactions
- **Responsive:** Mobile-friendly designs

## Example Usage

### Example 1: Interactive Dashboard
**You say:** "Create a dashboard artifact with multiple tabs showing charts, data tables, and filters"

**What happens:**
- Skill creates multi-component structure
- Implements tab navigation
- Adds charts and tables
- Creates filtering system
- Returns interactive artifact

### Example 2: Web Tool
**You say:** "Build a loan calculator with input fields, real-time calculations, and payment schedule visualization"

**What happens:**
- Skill creates form components
- Implements calculation logic
- Adds visualization
- Creates interactive experience

### Example 3: Mini-App
**You say:** "Make a task management app with add/edit/delete, filtering, and priority sorting"

**What happens:**
- Skill creates full app structure
- Implements CRUD operations
- Adds filtering and sorting
- Uses shadcn/ui components
- Returns functional mini-app
