# Internal Comms Skill - Company Communications

## What This Skill Does

Helps write all kinds of internal communications using company-preferred formats including status reports, leadership updates, newsletters, FAQs, incident reports, and project updates.

## When to Use This Skill

**Automatic triggers:**
- "status report", "leadership update", "company newsletter"
- Internal communication requests

**Use when you need:**
- Employee communications
- Team updates
- Leadership briefings
- Project status reports
- Incident communications
- Company newsletters
- FAQ documents
- Announcement drafts

## How to Invoke

**Explicit invocation:**
```
Use the internal-comms skill
```

**Trigger words (automatic):**
- "Write a status report for..."
- "Create a leadership update about..."
- "Draft a company newsletter..."
- "Write an incident report..."

## Capabilities

- **Status Reports:** Regular progress updates
- **Leadership Updates:** Executive communications
- **Newsletters:** Company-wide bulletins
- **Incident Reports:** Issue communications
- **FAQs:** Question and answer documents
- **Announcements:** Change and event notices
- **Project Updates:** Initiative progress reports
- **Team Communications:** Department messages

## Example Usage

### Example 1: Project Status Report
**You say:** "Write a status report for the Q2 platform migration project"

**What happens:**
- Skill structures status format
- Gathers key information
- Highlights progress and blockers
- Formats professionally
- Returns ready-to-send report

### Example 2: Leadership Update
**You say:** "Create a leadership update about our Q4 results and 2026 strategy"

**What happens:**
- Skill creates executive-level format
- Structures key messages
- Appropriate tone and detail level
- Returns polished update

### Example 3: Incident Communication
**You say:** "Draft an incident report for yesterday's service outage"

**What happens:**
- Skill structures incident format
- Covers timeline, impact, resolution
- Appropriate transparency level
- Returns communication draft

## Tips for Best Results

**Provide:**
- Target audience (all staff, leadership, team)
- Key information to communicate
- Tone preference (formal, casual, transparent)
- Any sensitive considerations
- Required format or template

**For best results:**
- Be clear about what happened/status
- Mention what readers need to know
- Specify any actions required

## Integration

**Works well with:**
- **docx skill:** Format as Word documents
- **pdf skill:** Create distribution PDFs
- **Email systems:** Draft email versions

## Common Use Cases

- Weekly status reports
- Quarterly business reviews
- Incident post-mortems
- Product launch announcements
- Organizational changes
- Policy updates

- Monthly newsletters
