# PPTX Skill - Microsoft PowerPoint Presentations

## What This Skill Does

Creates, reads, edits, and manipulates Microsoft PowerPoint (.pptx) presentations with professional layouts, speaker notes, and visual content.

## When to Use This Skill

**Automatic triggers:**
- "PowerPoint", "presentation", "slides", "deck"
- "pitch deck", "slide deck", "slideshow"
- Any .pptx file mention

**Use when you need:**
- Create presentation slides
- Edit existing slide decks
- Extract text from presentations
- Combine or split slide files
- Work with layouts and templates
- Add speaker notes

## How to Invoke

**Explicit invocation:**
```
Use the pptx skill
```

**Trigger words (automatic):**
- "Create a presentation about..."
- "Make me slides for..."
- "Build a pitch deck for..."
- "Generate a slideshow about..."

**File uploads:**
Upload a .pptx file to automatically activate this skill

## Capabilities

- **Create:** New presentations with professional layouts
- **Read:** Extract content, notes, layout information
- **Edit:** Modify slides while preserving design
- **Layouts:** Apply and customize slide layouts
- **Content:** Text, images, charts, tables, shapes
- **Notes:** Add and manage speaker notes
- **Themes:** Apply consistent visual themes
- **Master Slides:** Work with presentation templates

## Example Usage

### Example 1: Create Business Presentation
**You say:** "Create a 10-slide pitch deck for our SaaS product launch with problem, solution, market, traction, and ask slides"

**What happens:**
- Skill creates structured presentation
- Professional layout applied
- Key sections organized
- Visual hierarchy established
- Ready to present

### Example 2: Extract Content
**You say:** "Pull all the text from this presentation"
*(Upload .pptx file)*

**What happens:**
- Skill reads presentation
- Extracts text from all slides
- Includes speaker notes
- Returns organized content

### Example 3: Update Existing Deck
**You say:** "Update slide 5 with Q4 revenue data and add a growth chart"
*(Upload .pptx file)*

**What happens:**
- Skill opens presentation
- Locates slide 5
- Updates data
- Generates appropriate chart
- Returns updated file

## Tips for Best Results

**For creation:**
- Specify number of slides needed
- Describe desired content for each slide
- Mention visual style preferences
- Request speaker notes if needed
- Indicate any required charts or images

**For editing:**
- Reference specific slide numbers
- Be clear about what needs to change
- Preserve existing design unless instructed otherwise
