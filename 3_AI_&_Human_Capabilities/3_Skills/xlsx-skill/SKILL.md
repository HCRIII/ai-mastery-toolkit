# XLSX Skill - Microsoft Excel Spreadsheets

## What This Skill Does

Creates, reads, edits, and analyzes Microsoft Excel (.xlsx) spreadsheets with formulas, formatting, charts, and data analysis capabilities.

## When to Use This Skill

**Automatic triggers:**
- "Excel", "spreadsheet", ".xlsx", "data table"
- ".xlsm", ".csv", ".tsv" files
- Budget, financial model, data analysis requests

**Use when you need:**
- Create new spreadsheets
- Analyze existing data
- Compute formulas and calculations
- Format tabular data
- Create charts and visualizations
- Clean or restructure messy data
- Convert between tabular formats

## How to Invoke

**Explicit invocation:**
```
Use the xlsx skill
```

**Trigger words (automatic):**
- "Create a spreadsheet for..."
- "Analyze this data..."
- "Make me a budget tracker..."
- "Generate a financial model..."

**File uploads:**
Upload .xlsx, .xlsm, .csv, or .tsv files to automatically activate

## Capabilities

- **Create:** New spreadsheets with formulas and formatting
- **Read:** Extract data, formulas, formatting
- **Edit:** Modify cells, formulas, layouts
- **Calculate:** Complex formulas and functions
- **Format:** Cell styles, conditional formatting, number formats
- **Charts:** Create visual representations of data
- **Clean:** Fix messy data, remove junk, standardize format
- **Convert:** Between .xlsx, .csv, .tsv formats

## Example Usage

### Example 1: Create Budget Tracker
**You say:** "Create a monthly budget spreadsheet with categories for income, fixed expenses, variable expenses, and savings. Include formulas to calculate totals and remaining balance"

**What happens:**
- Skill creates structured spreadsheet
- Categories properly organized
- Formulas automatically calculated
- Formatting applied
- Ready to use

### Example 2: Analyze Data
**You say:** "Analyze this sales data and create a summary showing total revenue by product category"
*(Upload .xlsx file)*

**What happens:**
- Skill reads spreadsheet
- Aggregates data by category
- Creates summary table
- Generates chart if helpful
- Returns analysis

### Example 3: Clean Messy Data
**You say:** "This CSV has misplaced headers and junk rows - clean it up into a proper spreadsheet"
*(Upload .csv file)*

**What happens:**
- Skill identifies data structure
- Fixes headers
- Removes junk rows
- Standardizes format
- Returns clean .xlsx file

## Tips for Best Results

**For creation:**
- Describe desired structure and columns
- Specify any formulas or calculations needed
- Mention formatting preferences
- Request charts if you want visualizations

**For analysis:**
- Be specific about what insights you need
- Mention desired output format
- Request specific calculations or aggregations

**For cleaning:**
- Describe what's wrong with the data
- Specify desired final structure
