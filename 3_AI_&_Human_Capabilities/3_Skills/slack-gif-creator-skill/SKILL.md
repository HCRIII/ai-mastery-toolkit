# Slack GIF Creator Skill - Animated GIFs for Slack

## What This Skill Does

Creates animated GIFs optimized specifically for Slack with proper size constraints, validation tools, and animation concepts.

## When to Use This Skill

**Automatic triggers:**
- "GIF for Slack", "animated GIF"
- "make me a GIF of X doing Y for Slack"

**Use when you need:**
- Animated GIFs for Slack reactions
- Custom Slack emoji animations
- Team communication GIFs
- Animated responses

**Note:** Optimized for Slack's specific constraints (file size, dimensions, etc.)

## How to Invoke

**Explicit invocation:**
```
Use the slack-gif-creator skill
```

**Trigger words (automatic):**
- "Create a Slack GIF of..."
- "Make me an animated GIF for Slack..."
- "Generate a GIF showing..."

## Capabilities

- **Animated GIFs:** Create custom animations
- **Slack Optimization:** Proper sizing and constraints
- **Validation:** Ensures Slack compatibility
- **Animation:** Frame-by-frame or interpolated
- **Concepts:** Visual metaphors and representations
- **Reactions:** Expressive GIFs for team communication

## Tips for Best Results

**Specify:**
- Desired message or emotion
- Animation style (smooth, bouncy, subtle)
- Colors or branding
- Duration (short for reactions, longer for complex animations)

**For Slack optimization:**
- Keep file size in mind (Slack has limits)
- Consider mobile viewing
- Clear, simple visuals work best
