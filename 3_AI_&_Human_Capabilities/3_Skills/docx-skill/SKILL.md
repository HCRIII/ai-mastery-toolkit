# DOCX Skill - Microsoft Word Documents

## What This Skill Does

Creates, reads, edits, and manipulates Microsoft Word (.docx) documents with professional formatting, tracked changes, comments, and complex layouts.

## When to Use This Skill

**Automatic triggers:**
- "Word doc", "word document", ".docx"
- "report", "memo", "letter", "template"
- Professional documents with formatting
- Documents with tables of contents, headings, page numbers
- Letterheads and branded documents

**Use when you need:**
- Professional document creation
- Extract or reorganize content from Word files
- Insert or replace images in documents
- Find-and-replace in Word files
- Work with tracked changes or comments
- Convert content to polished Word format

## How to Invoke

**Explicit invocation:**
```
Use the docx skill
```

**Trigger words (automatic):**
- "Create a Word document about..."
- "Make me a report on..."
- "Write a memo for..."
- "Generate a professional document..."

**File uploads:**
Upload a .docx file to automatically activate this skill

## Capabilities

- **Create:** New documents with professional formatting
- **Read:** Extract text, tables, comments, tracked changes
- **Edit:** Modify existing documents while preserving formatting
- **Format:** Headers, footers, page numbers, tables of contents
- **Style:** Custom fonts, colors, spacing, margins
- **Tables:** Create and format complex tables
- **Images:** Insert, position, and size images
- **Track Changes:** Work with revision history
- **Comments:** Add, read, and respond to comments

## Tips for Best Results

**For document creation:**
- Specify desired length and structure
- Mention formatting requirements (fonts, spacing, margins)
- Indicate if you need headers, footers, page numbers
- Request table of contents if document is long
- Specify image placement preferences

**For editing:**
- Clearly state what needs to change
- Mention if you want to preserve certain formatting
- Indicate if tracked changes should be accepted/rejected

**For extraction:**
- Be specific about what information you need
- Mention desired output format
