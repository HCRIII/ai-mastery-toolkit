# Canvas Design Skill - Visual Art Creation

## What This Skill Does

Creates beautiful visual art in .png and .pdf formats using design philosophy and principles for posters, designs, and static visual pieces.

## When to Use This Skill

**Automatic triggers:**
- "poster", "design", "visual art", "piece of art"
- "create a design", "make art"

**Use when you need:**
- Original visual designs
- Posters and promotional materials
- Artistic compositions
- Static visual pieces
- Design concepts and mockups

**Note:** Creates original art - never copies existing artists' work to avoid copyright violations.

## How to Invoke

**Explicit invocation:**
```
Use the canvas-design skill
```

**Trigger words (automatic):**
- "Create a poster for..."
- "Design a visual for..."
- "Make me art about..."
- "Generate a design showing..."

## Capabilities

- **Create:** Original visual art and designs
- **Design Philosophy:** Applies professional design principles
- **Formats:** .png and .pdf outputs
- **Styles:** Various artistic and design styles
- **Composition:** Balanced layouts and visual hierarchy
- **Colors:** Intentional color schemes
- **Typography:** Professional text integration

## Example Usage

### Example 1: Event Poster
**You say:** "Create a poster for a jazz concert - modern, vibrant, with blue and gold colors"

**What happens:**
- Skill applies design principles
- Creates original composition
- Uses specified color palette
- Integrates typography
- Returns .png or .pdf file

### Example 2: Concept Visual
**You say:** "Design a visual representing growth and innovation for a tech startup"

**What happens:**
- Skill interprets concepts visually
- Creates metaphorical composition
- Applies professional design
- Outputs clean visual

### Example 3: Promotional Material
**You say:** "Make a promotional design for a sustainability workshop with earth tones and nature elements"

**What happens:**
- Skill creates thematic design
- Incorporates nature motifs
- Uses earth-tone palette
- Professional layout

## Tips for Best Results

**Be specific about:**
- Purpose and audience
- Desired mood or feeling
- Color preferences
- Style preferences (modern, classic, minimalist, bold)
- Key elements to include
- Output format (.png vs .pdf)

**For best designs:**
- Describe the message or concept clearly
- Mention any text that should appear
- Specify size or orientation if important
