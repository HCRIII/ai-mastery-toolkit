# Algorithmic Art Skill - Generative Art with Code

## What This Skill Does

Creates algorithmic art using p5.js with seeded randomness and interactive parameter exploration for generative art, flow fields, and particle systems.

## When to Use This Skill

**Automatic triggers:**
- "generative art", "algorithmic art"
- "flow fields", "particle systems"
- "creating art using code"

**Use when you need:**
- Art generated from algorithms
- Repeatable random patterns (seeded)
- Interactive art explorations
- Code-based visual systems
- Generative design patterns

**Note:** Creates original algorithmic art - never copies existing artists' work.

## How to Invoke

**Explicit invocation:**
```
Use the algorithmic-art skill
```

**Trigger words (automatic):**
- "Create generative art for..."
- "Make algorithmic art showing..."
- "Generate art using code..."
- "Create a flow field..."

## Capabilities

- **p5.js:** Uses p5.js framework for art creation
- **Seeded Randomness:** Reproducible random patterns
- **Interactive:** Parameter exploration and variations
- **Flow Fields:** Fluid, flowing visual patterns
- **Particle Systems:** Dynamic particle-based art
- **Algorithmic Patterns:** Code-generated designs
- **Variations:** Generate multiple versions from parameters

## Example Usage

**You say:** "Create a flow field art piece with blue and purple gradients, organic movement"

**What happens:**
