## How These Documents Work Together

**Manual** teaches the methodology → **You practice** using questions → **Tracker** captures discoveries → **SuperContext** documents patterns → **System Wisdom** matures the patterns → Better results through richer understanding → More discoveries → Repeat

The three-document system (Manual + Tracker + SuperContext) creates a compound learning loop. In v4, System Wisdom extends this loop with structured behavioral tracking and intelligence.
