# Human Behavioral Taxonomy

**Purpose:** Track the human actor's behavioral patterns, cognitive development, and collaborative skill evolution over time.

**What Is Tracked:**

- **Prompt Aptness** — Specificity, clarity, semantic precision, structural intelligence of requests
- **Context Provision** — Quality, depth, and organizational sophistication of context brought to interactions
- **Friction Response** — How productive friction moments are handled (Path A: immediate engagement vs. Path B: deferral)
- **Orchestration Skill** — Ability to manage concurrent threads, maintain strategic coherence, allocate cognitive load
- **Long-Term Value Awareness** — Recognition of friction as investment opportunity; building structure over solving instances
- **Communication Patterns** — Brevity signals, feedback specificity, correction style, collaborative register

**What Is NOT Tracked Here:**

- AI behaviors (see 2_AI_Behavioral_Taxonomy/)
- Interaction quality between the two actors (see 3_Dyadic_Dynamic_Taxonomy/)
- Task outputs or project state (see 4_Context_Files)

**Folder Contents:**

- `Current_State.md` — Present behavioral profile; loaded into AI context at session start
- `Comprehensive_Taxonomy.md` — Full map of all known human behavioral dimensions and their developmental levels
- `YYYY-MM-DD-HH-MM_Snapshot.md` — Timestamped captures of behavioral state at checkpoint moments
- `_Archive/` — Older snapshots stored for longitudinal analysis
