# System Wisdom

**Purpose:** Track the behavioral evolution of the human-AI collaborative system and the intelligence that emerges from it over time.

**Core Design Principle:** This system tracks three concurrent behavioral taxonomies — Human, AI, and the Shared Dynamic. The accumulated delta between timestamped snapshots is the primary measure of system development. A snapshot of this system at any moment is itself evidence of the CDPID framework's claims.

**What Lives Here (Behavioral Development):**

- **1_Human_Behavioral_Taxonomy/** — The human actor's behavioral patterns, cognitive development, and collaborative skill evolution
- **2_AI_Behavioral_Taxonomy/** — The AI actor's behavioral patterns, failure mode tendencies, and capability levels as experienced by this dyad
- **3_Dyadic_Dynamic_Taxonomy/** — The shared interaction dynamic, equilibration state, and collaborative performance of the dyad as a system

**What Emerges Here (System Learning):**

- **4_System_Intelligence/** — Accumulated patterns, validated principles, and optimization insights extracted from taxonomy data over time
- **5_Case_Evidence/** — Documented episodes that illustrate CDPID principles in action; research evidence and teaching material

**What Does NOT Live Here:**

- Operational coordination (project tracking, task management, dependencies) — lives in `4_Context_Files/`
- Context files (brand guidelines, client profiles, business strategy) — lives in `4_Context_Files/Level_0_Level_3/` (Intelligence Layer)
- Production workflow (asset creation, QC pipeline) — lives in `4_Context_Files/Level_4_Level_6/` (Production Layer)

**File Pattern Within Each Taxonomy Folder:**

- `README.md` — Why the folder exists and what it tracks
- `Current_State.md` — Loaded into AI context at session start; specifies present behavioral state
- `Comprehensive_Taxonomy.md` — Full map of all known behaviors for the actor (reference document)
- `YYYY-MM-DD-HH-MM_Snapshot.md` — Timestamped behavioral state captures (accumulating raw data)
- `_Archive/` — Older snapshots moved here to keep the active folder navigable

**Design Alignment:** This system is built in alignment with the CDPID research framework. It is both an operational tool and a research instrument. The structure is portable — usable by any human-AI dyad to track their own collaborative development.
