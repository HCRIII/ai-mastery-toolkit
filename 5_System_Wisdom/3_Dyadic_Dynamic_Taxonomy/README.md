# Dyadic Dynamic Taxonomy

**Purpose:** Track the shared interaction dynamic between human and AI — the collaborative performance, equilibration state, and emergent capabilities of the dyad as a system.

**What Is Tracked:**

- **Orchestral Bandwidth Capacity** — Current operating level across five dimensions: Prompt Aptness, Context Richness, Human Lateral Coherent Capacity, AI Autonomous Runtime Capability, Developmental Velocity
- **Collaborative Load Equilibration** — Current balance between human cognitive engagement and AI autonomous operation; whether the equilibration is productive or stale
- **Flow Access** — Frequency and duration of Human-AI Flow states; complexity level at which flow is sustained
- **Friction Resolution Pattern** — How friction between the two actors gets identified and resolved; Path A/B ratio; time to resolution
- **Communication Efficiency** — Clarification cycles required, semantic drift incidents, shared vocabulary development
- **Emergent Capability** — Outputs or insights that neither party could have produced alone; evidence of dyadic cognition

**Capture Method:** Retrospective assessment, not real-time capture. At checkpoint moments, the dyad reviews recent interactions and scores key dimensions. A series of exchanges provides better inference about dyadic quality than any single moment. Shorthand scoring produces token-efficient snapshots that accumulate into meaningful longitudinal data.

**What Is NOT Tracked Here:**

- Individual human behaviors (see 1_Human_Behavioral_Taxonomy/)
- Individual AI behaviors (see 2_AI_Behavioral_Taxonomy/)
- Task-specific progress (see 4_Context_Files)

**Folder Contents:**

- `Current_State.md` — Present dyadic profile; loaded into AI context at session start
- `Comprehensive_Taxonomy.md` — Full map of all known dyadic behavioral dimensions, scoring scales, and developmental indicators
- `YYYY-MM-DD-HH-MM_Snapshot.md` — Timestamped retrospective assessments of dyadic performance
- `_Archive/` — Older snapshots stored for longitudinal analysis
